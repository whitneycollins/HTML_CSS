Summary: Latitude vs. X

The purpose of this project was to analyze any weather patterns associated with latitudinal coordinates.

Using data form OpenWeatherMap API, 500 randomly selected cities are pulled to assemble a dataset to be used for analysis. Then, using this data, plots are created considering the following factors:

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (MPH) vs. Latitude
